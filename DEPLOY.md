# Deploy to GitHub and go live

Repo: **https://github.com/Nikulkumar-Dabhi/AI.prompt**

## 1. Push this project

```bash
cd "/Users/nikulsinhdabhi/Github/Prompt Master"
git remote set-url origin https://github.com/Nikulkumar-Dabhi/AI.prompt.git
git push -u origin main
```

## 2. Turn on GitHub Pages

1. On GitHub, open your repo → **Settings** → **Pages** (left sidebar).
2. Under **Source**, choose **Deploy from a branch**.
3. Branch: **main**, folder: **/ (root)**.
4. Click **Save**.

After a minute or two, the site will be at:

- **https://nikulkumar-dabhi.github.io/AI.prompt/**
